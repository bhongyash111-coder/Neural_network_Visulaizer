What is a Perceptron?

A perceptron is the simplest kind of neural network unit.
It takes some input values, multiplies each one by a weight,
adds them together, and then passes the result through an
activation function to decide the output.

output = activation( w‚ÇÅ¬∑x‚ÇÅ + w‚ÇÇ¬∑x‚ÇÇ + b )

It‚Äôs basically a tiny decision-maker. If the combined input is strong enough, it ‚Äúactivates.‚Äù If not, it stays inactive.

  • x‚ÇÅ: Is it sunny? (1 = yes, 0 = no)
  • x‚ÇÇ: Do you have free time? (1 = yes, 0 = no)

Your brain assigns importance (weights):

  • w‚ÇÅ = 0.8 (sun matters a lot)
  • w‚ÇÇ = 0.5 (free time matters moderately)

Then your brain combines the information:

weighted_sum = (w‚ÇÅ¬∑x‚ÇÅ) + (w‚ÇÇ¬∑x‚ÇÇ)
weighted_sum = (0.8 √ó x‚ÇÅ) + (0.5 √ó x‚ÇÇ)

Now apply a step function:

What is Forward Propagation?

Forward propagation is the process of passing the input values forward through the network to get the final output. Each neuron multiplies the input by weights, adds a bias, applies an activation function, and sends the result onward.

Input ‚Üí Weights ‚Üí Bias ‚Üí Activation ‚Üí Output

In simple words: It's the network's "thinking". It takes the inputs and calculates what it believes the answer should be.

Step 1: Inputs Enter the Network

In this step, our network receives two input values: IQ and CGPA. These inputs are passed forward along the connections into the neurons in the hidden layer. The softly glowing lines represent the signal traveling forward through the network.

Step 2: Weighted Inputs and Summation

Each connection has a weight, shown as W¬π‚ÇÅ‚ÇÅ, W¬π‚ÇÅ‚ÇÇ, W¬π‚ÇÇ‚ÇÅ, W¬π‚ÇÇ‚ÇÇ. Each hidden neuron multiplies the input values with their corresponding weights and sums them:

Œ£ ( W·µÄ √ó X + b )

Activation functions allow the network to learn non-linear patterns, which is what makes neural networks powerful.

  • ReLU (Rectified Linear Unit) ‚Äî Passes only positive signals. Simplifies learning. Most common in hidden layers.
  • Sigmoid ‚Äî Squashes values between 0 and 1. Useful when signals represent probabilities.
  • Tanh ‚Äî Squashes values between -1 and 1. Data is centered around zero, which can help learning.

In short, the hidden neurons learn features from the input data. They do not make the final decision ‚Äî they simply transform the information so that the output neuron can make a clearer decision later.

Step 3: Outputs from Hidden Layer Move to Output Neuron

Now, each hidden neuron has produced an output. These outputs are calculated as the weighted combination of iq and cgpa, plus a bias term for each neuron.

Neuron 1 Output = activation( W¬π‚ÇÅ‚ÇÅ ¬∑ iq + W¬π‚ÇÇ‚ÇÅ ¬∑ cgpa + b‚ÇÅ‚ÇÅ )
Neuron 2 Output = activation( W¬π‚ÇÅ‚ÇÇ ¬∑ iq + W¬π‚ÇÇ‚ÇÇ ¬∑ cgpa + b‚ÇÇ‚ÇÇ )

These two outputs now flow forward to the final neuron in the output layer.

Step 4: Final Output Calculation

The outputs from the two hidden neurons are now combined in the final output neuron. Just like before, the weighted values and a bias term are added:

≈∑ = activation( W¬≤‚ÇÅ‚ÇÅ ¬∑ O‚ÇÅ‚ÇÅ + W¬≤‚ÇÇ‚ÇÅ ¬∑ O‚ÇÅ‚ÇÇ + b‚ÇÇ‚ÇÅ )

Here:
O‚ÇÅ‚ÇÅ and O‚ÇÅ‚ÇÇ are the outputs from the two hidden neurons.
W¬≤‚ÇÅ‚ÇÅ and W¬≤‚ÇÇ‚ÇÅ are the weights from the hidden layer to the output neuron.
b‚ÇÇ‚ÇÅ is the bias for the output neuron.

If the problem is a Yes/No decision (like ‚ÄúPlacement or Not‚Äù), we apply a Step Function or Sigmoid at the output neuron:

  • Step Function ‚Üí If we need a hard final decision (0 or 1).
  • Sigmoid ‚Üí If we want a probability like ‚Äúthere is an 82% chance of placement‚Äù.

What is Backpropagation?

Backward propagation ‚Äî or backpropagation ‚Äî is how a neural network learns from its mistakes.

When the network makes a wrong prediction, it doesn‚Äôt just say ‚Äúoops.‚Äù It traces the error backward through all the layers to see which weights and connections caused the mistake, and then adjusts them a little to do better next time. 

You can think of it like this:

        => Forward propagation is ‚Äúthinking.‚Äù
        => Backward propagation is ‚Äúlearning from being wrong.‚Äù

Scene 1 ‚Äî Loss Computation

The network made a prediction ≈∑ and we compare it with the true value y. The difference between them is the loss, which tells us how wrong the prediction was. To Calculate the loss we need to choose a loss function for eg., MSE(y - ≈∑)¬≤.

True Label (y): 1

Predicted (≈∑): 0.42

Loss increases as the prediction moves away from the true value.

Scene 2 ‚Äî Backward Pass (Error Flow)

-> We want to reduce the Loss = (y ‚àí ≈∑)¬≤.
-> To reduce this loss, the network must adjust ≈∑.
-> But ≈∑ depends on the outputs of the two hidden neurons (O‚ÇÅ‚ÇÅ and O‚ÇÅ‚ÇÇ).

-> And these hidden neuron outputs depend on their weights and biases:

  • O‚ÇÅ‚ÇÅ depends on W¬π‚ÇÅ‚ÇÅ, W¬π‚ÇÇ‚ÇÅ and b‚ÇÅ‚ÇÅ
  • O‚ÇÅ‚ÇÇ depends on W¬π‚ÇÅ‚ÇÇ, W¬π‚ÇÇ‚ÇÇ and b‚ÇÅ‚ÇÇ
So the error first flows backward from the output neuron to the hidden neurons. Each hidden neuron receives a portion of the blame proportional to how much its output influenced the final prediction (through its weights).

Scene 3 ‚Äî Weights & Bias Update By Gradient Descent

To update a weight, the network checks two things: (1) How large the error was, and (2) How strong the input signal on that connection was. This gives the gradient, which tells the network how much that weight affected the mistake.

Gradient Derivation (Backpropagation)

Loss:

\[ L = (y - \hat{y})^2 \]

Output layer:The ≈∑ depends on the weights like: W¬≤‚ÇÅ‚ÇÅ,W¬≤‚ÇÇ‚ÇÅ & b‚ÇÇ‚ÇÅ so,

\[ \hat{y} = W^{2}_{11}\,O_{11} + W^{2}_{21}\,O_{12} + b_{21} \] \[ \frac{\partial L}{\partial \hat{y}} = -2\,(y - \hat{y}), \qquad \frac{\partial \hat{y}}{\partial W^{2}_{11}} = O_{11}, \; \frac{\partial \hat{y}}{\partial W^{2}_{21}} = O_{12}, \; \frac{\partial \hat{y}}{\partial b_{21}} = 1 \] Hence, \[ \frac{\partial L}{\partial W^{2}_{11}} = -2\,(y - \hat{y})\,O_{11}, \qquad \frac{\partial L}{\partial W^{2}_{21}} = -2\,(y - \hat{y})\,O_{12}, \qquad \frac{\partial L}{\partial b_{21}} = -2\,(y - \hat{y}) \]

Hidden layer (for \(O_{11}\)):

\[ O_{11} = a\!\left(W^{1}_{11}\,iq + W^{1}_{21}\,cgpa + b_{11}\right) \] \[ \frac{\partial L}{\partial W^{1}_{11}} = -2\,(y - \hat{y})\, W^{2}_{11}\, iq, \qquad \frac{\partial L}{\partial W^{1}_{21}} = -2\,(y - \hat{y})\, W^{2}_{11}\, cgpa, \qquad \frac{\partial L}{\partial b_{11}} = -2\,(y - \hat{y})\, W^{2}_{11} \]

\[ \frac{\partial L}{\partial W^{1}_{12}} = -2\,(y - \hat{y})\, W^{2}_{21}\, iq, \qquad \frac{\partial L}{\partial W^{1}_{22}} = -2\,(y - \hat{y})\, W^{2}_{21}\, cgpa, \qquad \frac{\partial L}{\partial b_{12}} = -2\,(y - \hat{y})\, W^{2}_{21} \]

Update rule:

\[ W_{\text{new}} = W_{\text{old}} - \eta \, \frac{\partial L}{\partial W_{\text{old}}}, \qquad b_{\text{new}} = b_{\text{old}} - \eta \, \frac{\partial L}{\partial b_{\text{old}}} \]
